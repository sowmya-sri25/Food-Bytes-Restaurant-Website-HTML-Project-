# 🍽️ Food Bytes – Restaurant Website
## 📌 Project Overview

**Food Bytes** is a simple restaurant website built using **HTML5** as part of my web development learning journey.
The website showcases different food categories, menu items, restaurant information, multimedia content, and a customer feedback form.
This project helped me understand how to structure a complete web page using core HTML elements.

## 🚀 Features

* 🏠 Restaurant homepage with title and navigation
* 🍳 Food categories: **Breakfast, Lunch, Dinner**
* 🖼️ Image gallery displaying food items
* 📜 Restaurant description section
* 🎥 Embedded restaurant video
* 📋 Menu table with food images and prices
* 🎧 Customer review audio section
* 📝 Customer feedback form
* ⭐ Font Awesome icons for better UI

## 🛠️ Technologies Used

* **HTML5**
* **Font Awesome (Icons)**
* Basic HTML elements:
  * Lists
  * Tables
  * Forms
  * Images
  * Audio & Video
  * Marquee animation
